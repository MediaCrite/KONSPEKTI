Конструкция match-case в Python появилась в версии 3.10 и позволяет сопоставлять значение с образцом (pattern matching). Если выражение соответствует шаблону, то выполняются определённые действия. 
```py
match выражение:
    case образец_1: # действия, если совпадение с образцом_1
    case образец_2: # действия, если совпадение с образцом_2
    case _: # действия, если ни один из образцов не совпал (опционально)
**Некоторые понятия:**
* **Выражение** — это переменная или результат вычисления, который проверяется на соответствие образцам.
* **Образец** — описание структуры или значения, с которым сравнивается выражение.
* **\_** (подчёркивание) — универсальный шаблон, соответствующий любому значению. Используется в качестве блока «по умолчанию».
**Как работает сопоставление с образцом**:
1. Python проверяет значение выражения из match с каждым образцом в блоках case последовательно.
2. Если находится соответствие, выполняется код в теле этого блока.
3. Если ни один из образцов не подходит, выполняется блок case _, если он указан.
4. Внутри блока case можно использовать переменные, подстановки и дополнительные проверки.
**Некоторые области применения match-case**:
* **Обработка пользовательских данных**. Например, при работе с ответами от API или сложными словарями.
* **Разбор вложенных структур**. Сопоставление с образцом хорошо справляется с вложенными структурами, например, при обработке данных в виде дерева.
* **Обработка ошибок с различными уровнями детализации**. Использование match-case упрощает обработку исключений/ошибок с различными типами данных.
```
