Цикл for
- Принцип работы: перебирает элементы последовательности (списка, кортежа, строки и т. д.) и выполняет заданный блок кода для каждого элемента. 
```py
Синтаксис: for item in sequence: # Код, перебирающий каждый элемент последовательности. 
Перебор списка чисел и вывод каждого числа: numbers = [1, 2, 3, 4, 5] for number in numbers: print(number).
Перебор символов в строке: message = "Python" for char in message: print(char).
Особенности:
Цикл выполняется до тех пор, пока в последовательности остаются элементы. Как только все элементы будут обработаны, цикл завершается. 
Можно использовать функцию range() для выполнения кода определённое количество раз.
Цикл while
Принцип работы: выполняет блок кода, пока указанное условие остаётся истинным. 
Синтаксис: while условие: # блок кода # выполняется, пока условие истинно. 
Примеры:
Подсчёт чисел от 1 до 5 и вывод каждого числа: count = 1 while count <= 5: print(count) count += 1.
Обработка пользовательского ввода — цикл продолжается, пока не будет введено корректное значение.
Особенности:
Если условие в цикле while никогда не станет ложным, цикл будет выполняться бесконечно. Чтобы избежать этого, нужно обеспечить способ выхода из цикла.
Для более гибкого управления циклом используются операторы break (выход из цикла) и continue (переход к следующей итерации).
```
