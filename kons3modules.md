# Modules/Модули
- программы python содержат тысячи, десятки тысяч и сотни тысяч строк кода.
- в таких условиях одним файлом с кодом не обойдешься - его нужно разбивать на части
- с целью получения доступа к коду, который находится в других файлах, в python реализован механизм модулей
```py
my_module - имя
my_module.py - файл
```
- каждый модуль может содержать переменные, классы и функции
- имя модуля соответствует имени файла
```py
math.py - имя файл
math - имя модуля
```
# package/пакет
пакеты служат для объединения модулей в группы
```py
my_package - имя
каталог:
my_package/     - директория/папка
   __init__.py
   my_module_1.py
   my_module_2.py
   ...
```
пакет - набор взаимосвязанных модулей(пакет тоже является модулем), предназначенный для решения задач определенного класса некоторой предметной области
пакет представляет собой папку, в которой содержаться модули и другие пакеты и обязательный файл __init__.py, отвечающий за инициализацию пакета
```json
json/
   __init__.py
   decoder.py
   encoder.py
   scanner.py
   tool.py
```
именем пакета является название каталога
- может быть импортирован
- может быть многократно использован

# Стандартная библиотека
- os - множество функций для работы с операционной системой
- random - функции для генерации случайных чисел, букв, случайного выбора элементов последовательности
- re - модуль для работы с регулярными выражениями
- collections - ппредоставляет специализированные типы данных - счетчики, очереди, упорядоченные словари
# Сторонние модули
```py
PyPI - главный каталог библиотек на Python. Содержит более 200 000 пакетов:
requests(http-клиент для Питона)
numpy(вычисления с многомерными массивами)
flask(пакет для создания веб-приложений)
```
- при импорте модуля Python выполняет весь код в нем
- в импорте пакета Python выполняет код в файле пакета __init__.py, если такой имеется
- все объекты, определенные в модуле или __init__.py, становятся доступны импортирующему коду
```py
import <модуль/пакет_1> [as <псевдоним_1>] [<модуль/пакет_2> as <псевдоним_2>]
```
- после импортирования модуля его название становится переменной, через которую можно получить доступ к атрибутам модуля
- если название модуля слишком длинное, то для него можно создать псевдоним с помощью ключевого слова as
- при использовании as доступ будет осуществлятся ТОЛЬКО с помощью переменной-псевдонима
```py
import time
import random
import os as windows_os
import math as m
print(time.time())
print(random.random())
print(windows_os.getcwd())
print(m.e)
```
from...import чаще всего используется для частичного импорта
```py
from math import pi as my_pi, e as my_e
from <модуль> import <атрибут_1> as <псевдоним_1>, <атрибут_2> as <псевдоним_2>
from time import localtime as ltime
from math import*  - лучше не использовать
```
# исполняемая часть модуля
